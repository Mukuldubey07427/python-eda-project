# python-eda-project
End-to-End Exploratory Data Analysis (EDA) of a retail sales dataset using Python, Pandas, NumPy, and Matplotlib in Google Colab. Includes data cleaning, feature engineering, statistical analysis, outlier detection, correlation analysis, and business visualizations.

# 📊 Retail Sales Exploratory Data Analysis

## Overview
This project performs an end-to-end Exploratory Data Analysis (EDA) on a retail sales dataset downloaded from Kaggle using Python in Google Colab.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib

## Project Workflow

✔ Data Collection (Kaggle)

✔ Data Cleaning

✔ Missing Value Analysis

✔ Duplicate Removal

✔ Feature Engineering

✔ Statistical Analysis

✔ Outlier Detection

✔ Correlation Analysis

✔ Data Visualization

## Visualizations

- Monthly Sales Trend
- Top Selling Products
- Sales Distribution
- Sales vs Profit
- Category-wise Sales
- Sales Representative Performance
